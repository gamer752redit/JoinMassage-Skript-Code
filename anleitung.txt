    How to Install the Minecraft Join Message Skript 


1. Download the Skript
    You can also find the skript on our GitHub.

2. Download the "Skript" Plugin
    -Get it here: https://www.spigotmc.org/resources/skript.114544/

3. Start Your Server

4. Move the Downloaded Skript (join_skript.sk) into the Plugin Folder
    -Navigate to: plugins > Skript > scripts > join_skript.sk

5. Change "SERVERNAME" to your server name

6. Reload the Skript to Start Using It!
    -Run the command: /skript reload join_skript.sk
